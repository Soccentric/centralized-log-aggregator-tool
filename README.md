# Centralized Log Aggregator with Smart Filtering

A lightweight daemon that collects logs from multiple sources (syslog, application logs, kernel messages), applies intelligent filtering based on severity/patterns, and stores them efficiently with rotation. Add a simple web UI or CLI for searching and real-time tailing.

## Getting Started

### Prerequisites
- Xilinx Vivado/Vitis toolchain
- ZUBoard development board

### Building the Project
```bash
source source_me.vivado
make
```
